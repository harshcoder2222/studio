# **App Name**: Universe Game Market

## Core Features:

- Item Listings: Display items with images, names, game, category, rarity, and price.
- Advanced Search & Filtering: Enable search by item/game name and filtering by game category, item type, rarity, and price range.
- Game-Specific Pages: Dedicated pages for each game, showing banner, logo, items, gamepasses, description, and stats.
- Wishlist/Favorites System: Allow users to save items to a wishlist for later purchase or tracking.
- Trending item Analyzer: Monitor real time marketplace transactions in order to predict which items will become trending, and when. This information will be used as a tool to promote items.
- Admin Dashboard: Hidden admin panel to manage game categories, games, products, pricing, and enable/disable items. Gamepasses will be sold for robux.
- Item Redirection: When an item is clicked, redirect to the associated Roblox gamepass link specified in the admin panel.
- User Authentication: Secure user accounts and marketplace transactions using encrypted data

## Style Guidelines:

- Primary color: Saturated blue-violet (#9400D3) to convey a sense of futuristic gaming and digital assets.
- Background color: Dark desaturated blue-violet (#231532) to emphasize content.
- Accent color: Bright lime green (#32CD32) for highlighting interactive elements and neon-like effects.
- Font Pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body.
- Use neon-style icons related to game categories and item types.
- Modern card-based design with rounded corners and subtle shadows for depth.
- Subtle hover animations and micro-interactions to enhance user engagement.
